# 初步设想

## 思想

## 			着重与交流的团队协作程序，通过创建一个团队工作室，成员上传工作文件并对工作文件进行修改。成员间可通过语音通话方式对修改的文件内容进行解释以达到工作任务的高效对接。

## **技术实现**

​	前端：html css/sass js React  { Yjs(协同编辑) WebSocket(实时通信) }

​	后端：java  SpringBoot
